# Activity Types Test Course

This course is designed for testing and development purposes. Each lecture demonstrates different activity types available in the LMS.

## Course Structure

- **Lecture 1**: Reflection + Quiz
- **Lecture 2**: Code + Annotate
- **Lecture 3**: Socratic Dialogue + Experiment
- **Lecture 4**: Simulation Activity

## Testing Guidelines

1. Enroll in the course
2. Complete activities in each lecture
3. Verify submissions are saved
4. Check progress tracking in dashboard
5. Test AI integrations (Socratic, Code Review)
