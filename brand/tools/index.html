Tools

Machine Spirits is also a working learning platform and a set of experimental instruments — live, maintained, used in real teaching, but kept off the main path. This is the one public door to them. Everything beyond it sits behind a login; nothing here is sold, only listed so that it can be found.

Student dashboard

Enrolments, progress, and course materials, from a learner’s side. Open in the platform →

Instructor dashboard

Cohorts, enrolment, and grading, for whoever is running a course. Open in the platform →

Evaluation dashboard

The model-evaluation harness — the same measurements the public probes will draw from once they ship. Open in the platform →

Knowledge base

The structured store of notes and sources the courses are built on. Open in the platform →

Experiment lab

A workbench for running, recording, and comparing model experiments. Open in the platform →

Documentation

Platform notes for instructors and contributors. Open in the platform →

Other instruments — a dialogue simulator, a learning map, and a few smaller experiments — live inside the same platform and are reached from within it. None of this is the point of Machine Spirits; it is the workshop behind the argument, kept open.
