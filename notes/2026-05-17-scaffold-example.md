# Course note — scaffold example

Seed entry establishing the `notes/` content source — the **course-note**
type from WEBSITE-REFACTOR.md §H.2 (the user ruled a dedicated `notes/`
source over a `type:` frontmatter flag, consistent with CONVENTIONS.md's
"location and form determine state — no frontmatter flags").

A course-note is a short teaching artifact derived from a course (`course:`
names which one). The public URL is **path-derived** from the file:
`notes/2026-05-17-scaffold-example.md` → `/notes/2026-05-17-scaffold-example`
(bare-path; the optional `YYYY-MM-DD-` prefix sorts notes chronologically,
mirroring the essay form). `slug` is never authored.
