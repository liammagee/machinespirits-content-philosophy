Student Guide

Welcome to Machine Spirits LMS! This guide will help you navigate courses, complete activities, and track your academic progress.

Getting Started

Creating an Account

  1. Click Sign In in the navigation bar
  2. Select Register if you’re new
  3. Enter your email, password, and name
  4. Click Create Account

Your account gives you access to: - Course enrollment - Activity submissions - Progress tracking - Grade viewing

Signing In

  1. Click Sign In
  2. Enter your email and password
  3. Click Login

You’ll stay signed in until you log out or your session expires.

Navigating Courses

Finding Courses

From the Homepage: 1. Scroll to the Courses section 2. Browse available courses 3. Click on a course card to view details

From the Navigation: 1. Click Courses in the top navigation 2. View all available courses 3. Filter by category or search

Course Information

Each course page shows: - Title and description - Instructor name - Semester/term - Prerequisites (if any) - Learning objectives - Required readings - Lecture list

Enrolling in Courses

How to Enroll

  1. Navigate to a course page
  2. Click the Enroll button
  3. Confirm your enrollment

Once enrolled, you can: - Access all lectures - Complete activities - Track your progress - View your grades

Unenrolling

  1. Go to the course page
  2. Click Unenroll
  3. Confirm to remove yourself

Note: Unenrolling may affect your grade records.

Accessing Lectures

Finding Lectures

  1. Navigate to your enrolled course
  2. Lectures are listed by week
  3. Click a lecture to open it

Reading Lectures

Lectures contain: - Main content in readable format - Images and diagrams - Speaker notes (expandable) - Activities at the end

Navigation Tips

  • Use the sidebar to jump between sections
  • Click the back button to return to the course
  • Bookmark important lectures

Completing Activities

Activity Types

Machine Spirits offers various activity types:

Type Description
Reflection Written responses with word limits
Quiz Multiple choice, short answer, matching
Code Programming exercises with tests
Annotate Text annotation and analysis
Socratic AI-guided philosophical dialogue
Experiment Interactive simulation labs
Simulation Social dynamics exploration

Finding Activities

Activities appear: - At the end of each lecture - In your Dashboard (all activities view) - Sorted by due date urgency

Activity Status

Activities show colored indicators: - Green: Completed/Submitted - Amber: Due soon (within 3 days) - Red: Overdue - Gray: Not yet due

Activity Walkthroughs

Reflection Activities

  1. Read the prompt carefully
  2. Type your response in the text area
  3. Watch the word counter (stay within limits)
  4. Your draft auto-saves locally
  5. Click Submit when ready

Tips: - Take time to think before writing - Address all prompts if multiple are given - Proofread before submitting

Quiz Activities

  1. Read each question carefully
  2. Select or type your answer
  3. For multiple choice: click the option
  4. For short answer: type your response
  5. For matching: drag or click to match pairs
  6. Click Submit Quiz when done

Tips: - You may see instant feedback after submission - Some quizzes show correct answers - Review explanations to learn from mistakes

Code Activities

  1. Read the problem description
  2. Review the starter code
  3. Write your solution in the editor
  4. Click Run Code to test
  5. Check the Tests tab for results
  6. Use AI Review for feedback (optional)
  7. Click Submit when satisfied

Tips: - Use hints if you’re stuck (click “Hint”) - Run tests frequently - AI Review provides constructive feedback - Green tests = passing, Red = failing

Annotate Activities

  1. Read the source text carefully
  2. Select text by clicking and dragging
  3. Choose an annotation type:
    • Claim: Main arguments
    • Evidence: Supporting facts
    • Question: Things you wonder about
    • Connection: Links to other ideas
    • Insight: Your own realizations
  4. Optionally add a comment
  5. Click Add Annotation
  6. Repeat until you meet the minimum
  7. Complete the synthesis section
  8. Click Submit

Tips: - Meet the required annotation types - Quality over quantity in annotations - The synthesis ties your annotations together

Socratic Activities

  1. Read the topic and instructions
  2. Type your initial response
  3. The AI tutor will ask questions
  4. Respond thoughtfully
  5. Continue the dialogue (minimum turns required)
  6. Click Complete Dialogue when done

Tips: - Think deeply before responding - The AI challenges you to think harder - There are no “wrong” answers - The goal is exploration, not being “right”

Experiment Activities

  1. Read the experiment objectives
  2. View the simulation canvas
  3. Try different parameter presets
  4. Form a hypothesis for each trial
  5. Observe the behavior
  6. Record observations for each trial
  7. Submit when you’ve completed required trials

Tips: - Use the pause button to examine patterns - Adjust sliders to see parameter effects - Write detailed observations - Connect what you see to the concepts

Simulation Activities

  1. Read about the current scenario
  2. Watch the agent simulation
  3. Switch between scenarios:
    • Recognition, Alienation, Reification
    • Dialectic, Emergence, Custom
  4. Adjust parameters (if allowed)
  5. Record observations as you go
  6. Write a reflection connecting observations
  7. Submit your study

Tips: - Try multiple scenarios for comparison - Note how parameter changes affect behavior - Connect simulations to theoretical concepts

Tracking Your Progress

Dashboard

Access your dashboard by clicking your name in the navigation.

The dashboard shows: - Completion count: X/Y activities done - Points earned: Your score so far - Due soon: Upcoming deadlines - Overdue: Past-due items - Progress bar: Visual completion

Grade Summary

Expand “Grade Summary by Course” to see: - Letter grade for each course - Percentage score - Points breakdown - Activities completed vs. total

Activity List

The dashboard lists all activities: - Sorted by due date (most urgent first) - Color-coded by status - Click any activity to navigate to it

Due Dates

Understanding Due Dates

  • Due dates are calculated from lecture dates
  • dueOffset in days (e.g., 7 days after lecture)
  • Check the badge on each activity

Due Date Badges

Badge Meaning
“Due in X days” Upcoming deadline
“Due today” Submit today!
“Due soon” Within 3 days
“Overdue” Past deadline
“Completed” Already submitted

Late Submissions

Late submissions may be accepted but could affect your grade. Check with your instructor for the late policy.

Tips for Success

Time Management

  1. Check your dashboard regularly
  2. Address “due soon” items first
  3. Don’t wait until the last minute
  4. Set reminders for deadlines

Quality Work

  1. Read instructions completely
  2. Review lecture content before activities
  3. Take your time on reflections
  4. Use AI features to improve code

Getting Help

  1. Re-read lecture materials
  2. Use hints in code activities
  3. Review AI feedback carefully
  4. Contact your instructor for questions

Troubleshooting

Can’t Submit Activity

  • Check you’re signed in
  • Ensure you meet minimum requirements
  • Try refreshing the page
  • Check your internet connection

Submission Not Saving

  • Auto-save works for reflections/code
  • Check for the “Saved” indicator
  • Submissions save server-side after submit

Can’t See Activities

  • Verify you’re enrolled in the course
  • Check the lecture has activities
  • Try refreshing the page

Forgot Password

Contact your instructor or administrator for password reset assistance.

Getting Help

If you need additional help:

  1. Check this guide for common questions
  2. Review lecture content for activity context
  3. Use in-app hints and AI assistance
  4. Contact your instructor for course-specific questions
  5. Report bugs via GitHub issues
